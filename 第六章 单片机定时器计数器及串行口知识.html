第六章 单片机定时器/计数器及串行口

本章将深入学习单片机定时器/计数器的工作原理、编程方法以及串行口通信的基础知识和应用， 通过项目案例掌握定时器和串行口的实际应用技能。

定时器/计数器 串行口通信

定时器/计数器

基本概念

  • 定时器/计数器是单片机的重要组成部分，可用于精确计时、产生定时中断和计数外部事件
  • 8051单片机有两个16位定时器/计数器：T0和T1，每个定时器/计数器由两个8位寄存器组成
  • 定时器模式下，计数器对内部机器周期计数；计数器模式下，对外部输入脉冲计数
  • 定时器/计数器的核心是一个加1计数器，当计数值溢出时产生中断

工作原理

  • 定时器/计数器的核心是一个16位加1计数器，由THx和TLx两个8位寄存器组成
  • 定时器模式下，计数脉冲来自内部时钟，频率为晶振频率的1/12
  • 计数器模式下，计数脉冲来自外部输入引脚T0(P3.4)或T1(P3.5)
  • 当计数器计满溢出时，TFx标志位置1，可触发中断或作为查询标志

工作模式

  • 模式0：13位定时器/计数器，由THx的8位和TLx的低5位组成
  • 模式1：16位定时器/计数器，由THx和TLx组成完整的16位计数器
  • 模式2：自动重装初值的8位定时器/计数器，TLx作为计数器，THx作为初值寄存器
  • 模式3：仅适用于T0，将T0分成两个独立的8位计数器，T1停止工作

编程步骤

  1. 1 确定定时器/计数器的工作模式，设置TMOD寄存器
  2. 2 计算定时器初值，并写入THx和TLx寄存器
  3. 3 根据需要，设置中断允许寄存器IE和中断优先级寄存器IP
  4. 4 启动定时器/计数器，设置TCON寄存器中的TRx位
  5. 5 编写中断服务程序或查询程序处理定时器溢出事件

定时器初值计算 

初值计算公式：
初值 = 2^n - (定时时间 / 机器周期)

其中：
- n为计数器位数（模式0为13，模式1为16，模式2为8）
- 机器周期 = 12 / 晶振频率

例如：使用12MHz晶振，定时1ms，工作在模式1
机器周期 = 12 / 12MHz = 1μs
初值 = 65536 - (1000μs / 1μs) = 64536 = 0xFC18H
即：TH0 = 0xFCH，TL0 = 0x18H

单片机串行口通信

基本概念

  • 串行通信是指数据一位一位地按顺序传送的通信方式，具有传输线少、成本低等优点
  • 8051单片机内置一个全双工串行通信接口，可实现单片机之间或单片机与其他设备之间的通信
  • 串行通信分为同步通信和异步通信，8051单片机采用异步通信方式
  • 异步通信以帧为单位进行数据传输，每帧包含起始位、数据位、奇偶校验位和停止位

串行口结构

  • 8051串行口主要由发送缓冲器SBUF、接收缓冲器SBUF、串行控制寄存器SCON和波特率发生器组成
  • 发送缓冲器SBUF和接收缓冲器SBUF共用一个地址99H，通过读写指令区分
  • 串行控制寄存器SCON用于设置串行口的工作方式、接收/发送控制和状态标志
  • 波特率发生器通常由定时器T1或T2构成，用于控制串行通信的速率

工作方式

  • 方式0：同步移位寄存器方式，波特率固定为fosc/12，主要用于扩展I/O口
  • 方式1：8位异步通信方式，波特率可变，由定时器T1或T2控制
  • 方式2：9位异步通信方式，波特率固定为fosc/64或fosc/32
  • 方式3：9位异步通信方式，波特率可变，由定时器T1或T2控制

编程步骤

  1. 1 确定串行口的工作方式，设置SCON寄存器
  2. 2 如果工作在方式1或方式3，设置定时器T1或T2为波特率发生器
  3. 3 计算波特率初值，并写入定时器寄存器
  4. 4 根据需要，设置中断允许寄存器IE和中断优先级寄存器IP
  5. 5 启动定时器T1或T2（如果用作波特率发生器）
  6. 6 通过SBUF寄存器进行数据的发送和接收

波特率计算 

方式1和方式3的波特率计算公式：
波特率 = (2^SMOD / 32) × (定时器T1溢出率)

其中：
- SMOD是PCON寄存器的最高位
- 定时器T1溢出率 = fosc / [12 × (256 - TH1)]

例如：使用11.0592MHz晶振，波特率为9600bps，SMOD=0
9600 = (1 / 32) × [11059200 / (12 × (256 - TH1))]
解得：TH1 = 253 = 0xFDH

定时器/计数器项目案例

案例1：LED闪烁控制

使用定时器控制LED以1秒为周期闪烁

项目说明

本项目使用定时器T0产生500ms的定时中断，在中断服务程序中切换LED的状态，从而实现LED以1秒为周期闪烁。

硬件连接

  • LED阳极接P1.0引脚
  • LED阴极通过220Ω电阻接地 
; 使用定时器T0实现LED闪烁(1秒周期)
ORG 0000H
    LJMP MAIN        ; 跳转到主程序

ORG 000BH           ; T0中断服务程序入口
    LJMP TIMER0_ISR  ; 跳转到T0中断服务程序

ORG 0030H
MAIN:
    MOV SP, #60H     ; 设置堆栈指针
    
    ; 初始化定时器T0
    MOV TMOD, #01H    ; 设置T0为模式1(16位定时器)
    MOV TH0, #3CH     ; 设置定时器初值，定时50ms
    MOV TL0, #0B0H
    SETB ET0         ; 使能T0中断
    SETB EA          ; 开总中断
    SETB TR0         ; 启动T0
    
    MOV R7, #10       ; 50ms * 10 = 500ms
    
    ; 主循环
MAIN_LOOP:
    SJMP MAIN_LOOP   ; 无限循环

; T0中断服务程序
TIMER0_ISR:
    PUSH ACC         ; 保护现场
    PUSH PSW
    
    ; 重新加载定时器初值
    MOV TH0, #3CH
    MOV TL0, #0B0H
    
    DJNZ R7, EXIT_ISR ; 如果未到500ms，直接退出
    
    ; 500ms时间到，切换LED状态
    CPL P1.0
    MOV R7, #10       ; 重新加载计数初值
    
EXIT_ISR:
    POP PSW          ; 恢复现场
    POP ACC
    RETI             ; 中断返回

END

案例2：数码管动态显示

使用定时器实现数码管动态扫描显示

项目说明

本项目使用定时器T1产生2ms的定时中断，在中断服务程序中进行数码管的动态扫描，实现4位数码管显示0-9999的计数。

硬件连接

  • 数码管段选信号接P0口
  • 数码管位选信号通过74HC138译码器接P2.0-P2.2 
; 使用定时器T1实现数码管动态显示
ORG 0000H
    LJMP MAIN        ; 跳转到主程序

ORG 001BH           ; T1中断服务程序入口
    LJMP TIMER1_ISR  ; 跳转到T1中断服务程序

ORG 0030H
MAIN:
    MOV SP, #60H     ; 设置堆栈指针
    
    ; 初始化P0口为数码管段选输出
    MOV P0, #0FFH
    
    ; 初始化定时器T1
    MOV TMOD, #20H    ; 设置T1为模式2(自动重装)
    MOV TH1, #0F3H    ; 设置定时器初值，定时2ms
    MOV TL1, #0F3H
    SETB ET1         ; 使能T1中断
    SETB EA          ; 开总中断
    SETB TR1         ; 启动T1
    
    MOV R0, #0        ; 显示缓冲区指针
    MOV R1, #0        ; 计数值
    
    ; 主循环
MAIN_LOOP:
    ; 计数值加1(0-9999)
    INC R1
    MOV A, R1
    CJNE A, #100, SKIP_RESET
    MOV R1, #0        ; 计数值超过99，重置为0
SKIP_RESET:
    
    ; 将计数值拆分为4位数字，存入显示缓冲区
    MOV A, R1
    MOV B, #100
    DIV AB
    MOV 30H, A        ; 百位
    MOV A, B
    MOV B, #10
    DIV AB
    MOV 31H, A        ; 十位
    MOV 32H, B        ; 个位
    
    ; 延时
    MOV R7, #200
DELAY:
    MOV R6, #250
    DJNZ R6, $
    DJNZ R7, DELAY
    
    SJMP MAIN_LOOP   ; 无限循环

; T1中断服务程序(数码管扫描)
TIMER1_ISR:
    PUSH ACC         ; 保护现场
    PUSH PSW
    PUSH DPH
    PUSH DPL
    
    ; 关闭所有数码管
    MOV P2, #00H
    
    ; 根据当前扫描位，选择显示的数字
    MOV A, R0
    MOV DPTR, #SEG_TAB
    MOVC A, @A+DPTR
    MOV P0, A        ; 输出段选信号
    
    ; 选择当前扫描位
    MOV A, R0
    MOV P2, A        ; 输出位选信号
    
    ; 更新扫描位(0-3)
    INC R0
    MOV A, R0
    CJNE A, #4, SKIP_RESET_R0
    MOV R0, #0       ; 扫描完4位后，从头开始
SKIP_RESET_R0:
    
    POP DPL          ; 恢复现场
    POP DPH
    POP PSW
    POP ACC
    RETI             ; 中断返回

; 数码管段码表(共阴)
SEG_TAB:
    DB 3FH, 06H, 5BH, 4FH, 66H, 6DH, 7DH, 07H ; 0-7
    DB 7FH, 6FH, 77H, 7CH, 39H, 5EH, 79H, 71H ; 8-F

END

串行口项目案例

案例1：串口通信测试

单片机与PC之间的串口通信

项目说明

本项目实现单片机与PC之间的串口通信，单片机接收PC发送的数据，并将其原样返回。使用串口调试助手可以观察通信效果。

硬件连接

  • 单片机RXD(P3.0)接MAX232的TXD
  • 单片机TXD(P3.1)接MAX232的RXD
  • MAX232通过RS232接口连接到PC 
; 串口通信测试程序(波特率9600bps)
ORG 0000H
    LJMP MAIN        ; 跳转到主程序

ORG 0023H           ; 串口中断服务程序入口
    LJMP UART_ISR    ; 跳转到串口中断服务程序

ORG 0030H
MAIN:
    MOV SP, #60H     ; 设置堆栈指针
    
    ; 初始化串口
    MOV SCON, #50H    ; 设置串口为方式1(8位数据，可变波特率)
    MOV PCON, #00H    ; SMOD=0
    
    ; 初始化定时器T1(波特率发生器)
    MOV TMOD, #20H    ; 设置T1为模式2(自动重装)
    MOV TH1, #0FDH    ; 设置波特率为9600bps(晶振11.0592MHz)
    MOV TL1, #0FDH
    SETB TR1         ; 启动T1
    
    SETB ES          ; 使能串口中断
    SETB EA          ; 开总中断
    
    ; 发送欢迎信息
    MOV A, #'H'
    ACALL SEND_CHAR
    MOV A, #'e'
    ACALL SEND_CHAR
    MOV A, #'l'
    ACALL SEND_CHAR
    MOV A, #'l'
    ACALL SEND_CHAR
    MOV A, #'o'
    ACALL SEND_CHAR
    MOV A, #' '
    ACALL SEND_CHAR
    MOV A, #'W'
    ACALL SEND_CHAR
    MOV A, #'o'
    ACALL SEND_CHAR
    MOV A, #'r'
    ACALL SEND_CHAR
    MOV A, #'l'
    ACALL SEND_CHAR
    MOV A, #'d'
    ACALL SEND_CHAR
    MOV A, #13       ; 回车符
    ACALL SEND_CHAR
    MOV A, #10       ; 换行符
    ACALL SEND_CHAR
    
    ; 主循环
MAIN_LOOP:
    SJMP MAIN_LOOP   ; 无限循环

; 发送字符子程序
SEND_CHAR:
    MOV SBUF, A      ; 发送数据
WAIT_TX:
    JNB TI, WAIT_TX  ; 等待发送完成
    CLR TI           ; 清除发送完成标志
    RET

; 串口中断服务程序
UART_ISR:
    PUSH ACC         ; 保护现场
    PUSH PSW
    
    JNB RI, UART_EXIT ; 不是接收中断，直接退出
    
    CLR RI           ; 清除接收中断标志
    MOV A, SBUF      ; 读取接收到的数据
    
    ; 回显接收到的数据
    ACALL SEND_CHAR
    
UART_EXIT:
    POP PSW          ; 恢复现场
    POP ACC
    RETI             ; 中断返回

END

案例2：串口控制LED

通过串口发送指令控制LED

项目说明

本项目实现通过串口发送指令控制LED的亮灭。发送'A'点亮LED，发送'B'熄灭LED，发送'C'切换LED状态。

硬件连接

  • LED阳极接P1.0引脚
  • LED阴极通过220Ω电阻接地
  • 单片机串口通过MAX232连接到PC 
; 串口控制LED程序
ORG 0000H
    LJMP MAIN        ; 跳转到主程序

ORG 0023H           ; 串口中断服务程序入口
    LJMP UART_ISR    ; 跳转到串口中断服务程序

ORG 0030H
MAIN:
    MOV SP, #60H     ; 设置堆栈指针
    
    ; 初始化LED控制引脚
    SETB P1.0        ; 默认LED熄灭
    
    ; 初始化串口
    MOV SCON, #50H    ; 设置串口为方式1(8位数据，可变波特率)
    MOV PCON, #00H    ; SMOD=0
    
    ; 初始化定时器T1(波特率发生器)
    MOV TMOD, #20H    ; 设置T1为模式2(自动重装)
    MOV TH1, #0FDH    ; 设置波特率为9600bps(晶振11.0592MHz)
    MOV TL1, #0FDH
    SETB TR1         ; 启动T1
    
    SETB ES          ; 使能串口中断
    SETB EA          ; 开总中断
    
    ; 发送使用说明
    MOV A, #'C'
    ACALL SEND_CHAR
    MOV A, #'o'
    ACALL SEND_CHAR
    MOV A, #'m'
    ACALL SEND_CHAR
    MOV A, #'m'
    ACALL SEND_CHAR
    MOV A, #'a'
    ACALL SEND_CHAR
    MOV A, #'n'
    ACALL SEND_CHAR
    MOV A, #'d'
    ACALL SEND_CHAR
    MOV A, #':'
    ACALL SEND_CHAR
    MOV A, #' '
    ACALL SEND_CHAR
    MOV A, #'A'
    ACALL SEND_CHAR
    MOV A, #'-'
    ACALL SEND_CHAR
    MOV A, #'O'
    ACALL SEND_CHAR
    MOV A, #'N'
    ACALL SEND_CHAR
    MOV A, #'  '
    ACALL SEND_CHAR
    MOV A, #'B'
    ACALL SEND_CHAR
    MOV A, #'-'
    ACALL SEND_CHAR
    MOV A, #'O'
    ACALL SEND_CHAR
    MOV A, #'F'
    ACALL SEND_CHAR
    MOV A, #'F'
    ACALL SEND_CHAR
    MOV A, #'  '
    ACALL SEND_CHAR
    MOV A, #'C'
    ACALL SEND_CHAR
    MOV A, #'-'
    ACALL SEND_CHAR
    MOV A, #'T'
    ACALL SEND_CHAR
    MOV A, #'O'
    ACALL SEND_CHAR
    MOV A, #'G'
    ACALL SEND_CHAR
    MOV A, #'G'
    ACALL SEND_CHAR
    MOV A, #'L'
    ACALL SEND_CHAR
    MOV A, #'E'
    ACALL SEND_CHAR
    MOV A, #13       ; 回车符
    ACALL SEND_CHAR
    MOV A, #10       ; 换行符
    ACALL SEND_CHAR
    
    ; 主循环
MAIN_LOOP:
    SJMP MAIN_LOOP   ; 无限循环

; 发送字符子程序
SEND_CHAR:
    MOV SBUF, A      ; 发送数据
WAIT_TX:
    JNB TI, WAIT_TX  ; 等待发送完成
    CLR TI           ; 清除发送完成标志
    RET

; 串口中断服务程序
UART_ISR:
    PUSH ACC         ; 保护现场
    PUSH PSW
    
    JNB RI, UART_EXIT ; 不是接收中断，直接退出
    
    CLR RI           ; 清除接收中断标志
    MOV A, SBUF      ; 读取接收到的数据
    
    ; 判断接收到的命令
    CJNE A, #'A', CHECK_B
    ; 命令A：点亮LED
    CLR P1.0
    SJMP SEND_ACK
    
CHECK_B:
    CJNE A, #'B', CHECK_C
    ; 命令B：熄灭LED
    SETB P1.0
    SJMP SEND_ACK
    
CHECK_C:
    CJNE A, #'C', SEND_ACK
    ; 命令C：切换LED状态
    CPL P1.0
    
SEND_ACK:
    ; 发送确认信息
    MOV A, #'>'
    ACALL SEND_CHAR
    MOV A, #' '
    ACALL SEND_CHAR
    MOV A, SBUF      ; 回显接收到的命令
    ACALL SEND_CHAR
    MOV A, #' '
    ACALL SEND_CHAR
    MOV A, #'='
    ACALL SEND_CHAR
    MOV A, #' '
    ACALL SEND_CHAR
    JB P1.0, LED_OFF
    MOV A, #'O'
    ACALL SEND_CHAR
    MOV A, #'N'
    ACALL SEND_CHAR
    SJMP SEND_END
LED_OFF:
    MOV A, #'O'
    ACALL SEND_CHAR
    MOV A, #'F'
    ACALL SEND_CHAR
    MOV A, #'F'
    ACALL SEND_CHAR
SEND_END:
    MOV A, #13       ; 回车符
    ACALL SEND_CHAR
    MOV A, #10       ; 换行符
    ACALL SEND_CHAR
    
UART_EXIT:
    POP PSW          ; 恢复现场
    POP ACC
    RETI             ; 中断返回

END
